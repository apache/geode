Logging

GemFire provides a {@linkplain org.apache.geode.LogWriter logging API} that is similar to the one provided by standard java. The {@link org.apache.geode.internal.logging.LogWriterImpl} class provides implementations of a number of the LogWriter methods. There is one concrete implementation of LogWriterImpl. It is {@link org.apache.geode.internal.logging.PureLogWriter} that writes to a VM-local {@link java.io.PrintStream}.

The {@link org.apache.geode.internal.logging.MergeLogFiles} utility can be used to merged multiple log files together sorted by the timestamp of their entries.
